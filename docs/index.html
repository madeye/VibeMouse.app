VibeMouse

Mouse-side-button voice dictation for macOS. Bind speech‑to‑text to your mouse side buttons—talk, click, code.

macOS 13+ Python 3.10+ SenseVoice ONNX Apache-2.0
Get Started View on GitHub

How It Works

🖌

Front Button

Toggle voice recording on and off. Press once to start, press again to stop and transcribe.

⏎

Rear Button · Idle

Sends Enter (or Ctrl+Enter / Shift+Enter). Submit prompts, confirm dialogs—no keyboard needed.

⚡

Rear Button · Recording

Stops recording and dispatches the transcript to OpenClaw for AI‑assisted processing.

Features

SenseVoice ONNX ASR

Fast, accurate speech recognition running locally via ONNX Runtime. Hardware-accelerated on Apple Silicon via CoreML and the Apple Neural Engine.

Gesture Recognition

Swipe while holding a side button to trigger custom actions—workspace switching, record toggles, and more.

Smart Output Routing

Types text directly into the focused app. Falls back to clipboard or paste automatically with reason tracking.

OpenClaw Integration

Route transcripts to your local AI assistant with configurable agent, timeout, and retry settings.

Built-in Doctor

Run vibemouse doctor to diagnose permissions, audio input, OpenClaw connectivity, and config issues.

Zero External Services

All speech recognition runs on-device. No API keys, no cloud dependencies, no data leaves your Mac.

Quick Start

  • macOS 13+ (Ventura or later)
  • Python 3.10+
  • Accessibility permission for your terminal app
Install
python3 -m venv .venv && source .venv/bin/activate
pip install -U pip
pip install -e .

Speech recognition uses ONNX Runtime with Apple Neural Engine acceleration on Apple Silicon.

Run
vibemouse
Diagnose
vibemouse doctor          # check everything
vibemouse doctor --fix    # auto-fix then re-check

Configuration

All settings are environment variables. No config files needed.

VariableDefaultPurpose
VIBEMOUSE_ENTER_MODEenterRear-button submit: enter, ctrl_enter, shift_enter, none
VIBEMOUSE_GESTURES_ENABLEDfalseEnable gesture recognition
VIBEMOUSE_AUTO_PASTEfalseAuto-paste when output falls back to clipboard
VIBEMOUSE_PREWARM_ON_STARTtruePreload ASR model on startup
VIBEMOUSE_OPENCLAW_AGENTmainOpenClaw agent ID for transcript dispatch
VIBEMOUSE_FRONT_BUTTONx1Mouse button for record toggle
VIBEMOUSE_REAR_BUTTONx2Mouse button for enter / OpenClaw
